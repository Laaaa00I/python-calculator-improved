# Калькулятор и конвертер на Python (УЛУЧШЕННАЯ ВЕРСИЯ)

## Функции
- Основы арифметики: плюс, минус, умножить, разделить, возвести в степень, корень.
- Тригонометрия: синус, косинус, тангенс, котангенс, секанс, косеканс (вычисление в радианах).
- Автономный инструмент: Преобразователь радианы <-> градусы.
- Чистая модульная архитектура.
- Комментарии на русском и английском языках внутри кода

## Как запустить
1. Убедитесь, что установлен Python.
2. Запустите `main`, чтобы использовать калькулятор.
3. Запустите `Converter` для преобразования значений.
