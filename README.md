# Python Calculator & Converter (IMPROVED VERSION)

## Features
- Basic arithmetic: Sum, Sub, Mult, Div, Pow, Root.
- Trigonometry: Sin, Cos, Tan, Cot, Sec, Cosec (calculation in radians).
- Standalone tool: Radians <-> Degrees converter.
- Clean modular architecture.
- Comments in Russian and English inside the code

## How to run
1. Make sure Python is installed.
2. Run `main` to use the calculator.
3. Run `Converter` to convert values.
